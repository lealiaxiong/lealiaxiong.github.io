Raspberry Pi Environmental Monitor

Inspired by Adafruit IO Environmental Monitor for Feather or Raspberry Pi by Brent Rubell.

This project uses a Raspberry Pi Zero W to interface with a variety of sensors and display the data live. The Pi runs a Flask application that reads the data from the sensors, writes to a SQLite database, and displays the most recent data along with live-updating plots on a web page.

Sensors:

  • BME280

    • Temperature
    • Humidity
    • Barometric pressure

  • SGP30

    • Equivalent CO2
    • Total volatile organic compounds

  • VEML7700

    • Visible light

  • VEML6070

    • UV index
